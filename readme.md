# Quiero Viajes
Administrador de viajes empresariales con Google Maps para Wordpress.

## Instucciones

### API Key
Paso 1, crear API en google cloud con las siguientes bibliotecas
- Places API
- Places API (New)
- Directions API
- Distance Matrix API
- Maps JavaScript API

Paso 2, cargar la API Key en menu Admin de Wordpress Viajes > Ajustes
